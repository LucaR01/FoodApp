## Features
This app presents the following features:
- Buy foods.
- Add Foods to your favorite list.
- Possibility to save, check details and delete favorite foods in a database.
- Update in-app settings:
  - Theme
    - Light
    - Dark
  - Language
    - English
    - Italian
